# Личный проект «Way»

## Основные команды для работы
* Установка - `npm i`
* Запуск локального сервера - `npm start`
* Сборка проекта без запуска локального сервера - `npm run-script build`
* Запуск тестирования на соответствия код-гайдам - `npm test`
